    Class: DynamicView

    Defined in: src/mozart/dynamic-view.coffee
    Inherits: View

    Overview

    DynamicView is a View that will create a set of views depending on a supplied schema. The schema should be an array of strings or objects, with each object containing a viewClass to instantiate.

    DynamicView is intended to ease generation of forms given a dynamic schema.
