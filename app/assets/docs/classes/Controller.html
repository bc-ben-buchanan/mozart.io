    Overview

    Controller is the base class for Mozart Controllers.

    Caveat: Mozart's interpetation of controllers as a concept is quite different from other frameworks that adopt the MVC/MVVM/MV* paradigm.

    Please see http://www.mozart.io/guides/top_controller_pattern for more info.
