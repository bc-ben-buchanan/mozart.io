    Class: OneToManyPolyCollection

    Defined in: src/mozart/model-onetomanypolycollection.coffee
    Inherits: InstanceCollection

    Overview

    OneToManyPolyCollection is a relation class representing a scope of model instances in a polymorphic one-to-many (hasManyPoly) relation. Instances of this class are returned by relation methods created by the hasManyPoly method on Mozart.Model

    Please note that relation class do not actually contain lists of Instances. The relation class instance provides a set of scoped queries for the relation from the point of view of the Instance that created it, having much the same interface as Mozart.Model allowing its use in collections, etc.

    For more information, see http://www.mozart.io/guides/understanding_relations
