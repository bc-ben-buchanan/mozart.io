    Class: Collection

    Defined in: src/mozart/collection.coffee
    Inherits: View

    Overview

    Collection is the the view class for the Mozart collection handlebars helper - a view that renders a collection of other views, one for each data item in its collection property.

    The collection property can be: A Mozart Model A Mozart Relation object * An Array of objects

    If an array is supplied, collection expects an array of objects, each of which must have a at least one unique property, specified by @idField.

    For more information, see http://mozart.io/collection_demo
