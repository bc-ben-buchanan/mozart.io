    Class: Instance

    Defined in: src/mozart/model-instance.coffee
    Inherits: MztObject

    Overview

    Instance represents a single item in a Model. It is conceptually equivalent to a database row.

    Instances should not be created directly - use the initInstance or createFromValues methods on the model itself
